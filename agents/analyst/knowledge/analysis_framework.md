# Структура анализа конкурентов кофейни

## Исходные данные
- **Наша кофейня**: 3 точки в Алматы, средний чек 2500 ₸

## План анализа

### 1. Сравнительный анализ ценовой политики
- Средний чек по конкурентам
- Диапазон цен на основные позиции (эспрессо, капучино, латте, десерты)
- Позиционирование (премиум/средний/эконом сегмент)
- Программы лояльности и скидки

### 2. Оценка маркетинговых стратегий
- Присутствие в социальных сетях (Instagram, Facebook, TikTok)
- Активность и охваты
- Уникальные торговые предложения (УТП)
- Рекламные кампании
- Партнерства и коллаборации

### 3. Сильные и слабые стороны конкурентов
Для каждого игрока:
- **Сильные стороны**: что делают хорошо
- **Слабые стороны**: уязвимые места, пробелы

### 4. SWOT-анализ нашей позиции
- **Strengths** (Сильные стороны)
- **Weaknesses** (Слабые стороны)
- **Opportunities** (Возможности)
- **Threats** (Угрозы)

### 5. Рекомендации
Конкретные действия для повышения конкурентоспособности:
- Краткосрочные (1-3 месяца)
- Среднесрочные (3-6 месяцев)
- Долгосрочные (6-12 месяцев)

## Ожидаемые данные от researcher
Для каждого из топ-5 конкурентов:
1. Название и количество точек
2. Средний чек
3. Ценовой диапазон основных позиций
4. Маркетинговая активность
5. Целевая аудитория
6. Особенности позиционирования
7. Отзывы клиентов / репутация
8. Дополнительные услуги
