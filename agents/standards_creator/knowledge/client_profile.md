# Профиль клиента

Компания: сеть кофеен "АлматыКофе".
Рынок: Казахстан (Алматы, Астана, Шымкент), 3 филиала.
Языки: русский и казахский.
Сегмент: средний, кофе навынос + посадка.
Ценности: дружелюбие, скорость, чистота, уважение к сотрудникам.
Бюджет на изменения: ограниченный (небольшая сеть, не корпорация).

Цель собственника: поднять качество обслуживания до уровня мировых
сетей, но реалистично для сети из 3 точек.
